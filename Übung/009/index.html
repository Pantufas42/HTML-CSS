Códigos e citações

A tag code serve para colocar na fonte mono. O que ajuda a ler o código, mas não deixa totalmente fiel ás normas de escrita. Exemplo:

from time import sleep print('Bem vindo ao meu contador!') input() a = 0 while a != 10: sleep(1) a = a + 1 print(a)

A tag pre resolve o problema ela significa pré formatado(ou seja o HTML não vai formatar nada e vai colocar o código de jeito que você escreveu). Lembre-se de colocar a tag code DENTRO da tag pre.


from time import sleep
print('Bem vindo ao meu contador!')
input()
a = 0
while a != 10:
    sleep(1)
    a = a + 1
    print(a)
